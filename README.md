# logiczoo

A calculator for logic.

## Installing:

 - Install [Stack](https://docs.haskellstack.org/en/stable/README/), the Haskell tool
 - Clone this repo
 - `cd` into the repo
 - `$ stack install`
 - `$ logiczoo-exe`

## Examples:

 - To view the truth table for a sentence: `logiczoo-exe truthtable --sentence "~(A&B)"`
